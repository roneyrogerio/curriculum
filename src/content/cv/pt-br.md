- Deploy automatizado por GitHub Actions em pushes para `main`, com build e publicação de imagem multi-arquitetura (`linux/amd64` e `linux/arm64`) no GitHub Container Registry.
- Entrega em container multi-stage: build da aplicação em `node:20-alpine` e publicação dos arquivos estáticos via `nginxinc/nginx-unprivileged:1.27-alpine`, servindo na porta `8080`.
- Execução em Knative Service sobre Kubernetes (Oracle Kubernetes Engine), com namespace dedicado, `imagePullSecret` para GHCR e mapeamento do domínio customizado `roneyrogerio.dev`.
- Equipe e operação: projeto individual, cobrindo arquitetura, implementação, experiência visual, containerização, CI/CD e configuração de deploy.
